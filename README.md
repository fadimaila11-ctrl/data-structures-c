# Data Structures in C

This project contains implementations of fundamental data structures in C.

## Implemented Structures

### Stack
- push()
- pop()
- peek()
- isEmpty()

### Queue (Linked List based)
- enqueue()
- dequeue()

### Singly Linked List
- insert
- delete
- traversal

## Concepts Covered
- Pointers
- Dynamic Memory Allocation (malloc, free)
- Modular programming (.h / .c separation)

## Purpose
This project was built to strengthen my understanding of low-level programming and core data structures in C.
